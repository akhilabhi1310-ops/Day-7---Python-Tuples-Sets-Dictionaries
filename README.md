# 💡 Key Concepts Learned

## Tuple

A Tuple is an ordered and immutable collection of elements. Once created, its elements cannot be modified. Tuples allow duplicate values and support indexing and slicing.

### Advantages

- Faster than lists
- Immutable
- Memory efficient
- Supports indexing
- Can store multiple data types

---

## Set

A Set is an unordered collection of unique elements. Duplicate values are automatically removed. Sets are mainly used for mathematical operations and checking uniqueness.

### Advantages

- Removes duplicate values
- Fast searching
- Efficient mathematical operations
- Easy membership testing

---

## Dictionary

A Dictionary stores data as key-value pairs. Each key is unique and is used to access its corresponding value efficiently.

### Advantages

- Fast data retrieval
- Easy to update
- Flexible data storage
- Supports nested structures
- Useful in real-world applications

---

# 📊 Comparison

| Feature | List | Tuple | Set | Dictionary |
|----------|------|--------|-----|------------|
| Ordered | ✅ | ✅ | ❌ | ✅ |
| Mutable | ✅ | ❌ | ✅ | ✅ |
| Duplicate Values | ✅ | ✅ | ❌ | Values Only |
| Indexing | ✅ | ✅ | ❌ | Using Keys |
| Syntax | [] | () | {} | {key:value} |
